## Objective

The objective of this task is to solve a **business optimization problem** using **Linear Programming** with the PuLP library. The goal is to determine the optimal production strategy that maximizes profit while considering limited resources.

---

## Technologies Used

* Python
* PuLP (Linear Programming Library)

---

## Problem Statement

A factory produces two products:

* Product A
* Product B

Each product requires different amounts of labor and material resources. Since resources are limited, the objective is to determine the optimal quantity of each product that should be produced to **maximize total profit**.

---

## Optimization Approach

The following steps are used to solve the problem:

1. Define decision variables (number of units of each product)
2. Create an objective function to maximize profit
3. Define constraints based on available labor and material
4. Solve the linear programming model using PuLP

---

## Model Implementation

The optimization model includes:

* Decision variables
* Objective function
* Resource constraints
* Solver execution

The PuLP library is used to compute the optimal solution.

---

## Expected Output

The model calculates:

* Optimal number of Product A units
* Optimal number of Product B units
* Maximum achievable profit
